Guía Profesional: Configuración de VPS en Oracle Cloud con PHP y MySQL

Bienvenido a esta guía Basica completa sobre cómo configurar una cuenta en Oracle Cloud, conectarte a tu VPS, crear carpetas para alojar tu sitio web, utilizar herramientas como netstat, acceder a phpMyAdmin, practicar en OverTheWire y utilizar un script PHP para guardar datos en MySQL.

1. Crear cuenta en Oracle Cloud

Accede a https://cloud.oracle.com/, regístrate gratuitamente y crea una instancia desde la sección Compute Instances.

2. Crear carpeta dentro de /var/www/html

ssh usuario@tu_ip_vps
sudo mkdir -p /var/www/html/misitio
sudo chown -R $USER:www-data /var/www/html/misitio

3. Verificar puertos abiertos con netstat

sudo netstat -tulpn

4. Acceder a phpMyAdmin

Abre tu navegador y visita http://tu_ip/phpmyadmin. Ingresa tus credenciales para acceder a tu base de datos.

5. Aprende en OverTheWire

Fortalece tus conocimientos en comandos Linux y seguridad: OverTheWire - Bandit

6. Código PHP para insertar datos en MySQL

<?php
$conexion = new mysqli("localhost", "usuario", "clave", "basedatos");
if($conexion->connect_error){
  die("Error: " . $conexion->connect_error);
}
$nombre = $_POST['nombre'];
$correo = $_POST['correo'];
$sql = "INSERT INTO registro (nombre, correo) VALUES ('$nombre', '$correo')";
if($conexion->query($sql) === TRUE){
  echo "Registro exitoso";
}else{
  echo "Error: " . $conexion->error;
}
$conexion->close();
?>
